API Documentation

API 1: Get Available Qualities

        
            Request:
            {
                "key": "api key",
                "video_link": "video link"
            }
            
            Response:
            {
                "status": status_code,
                "title": "video title",
                "available_video_resolutions": ["res1", "res2"],
                "available_audio_qualities": ["qual1", "qual2"]
            }

API 2: Download Video

        
            Request:
            {
                "key": "api key",
                "video_link": "video link",
                "resolution": "resolution"
            }
            
            Response:
            {
                "status": status_code,
                "title": "video title",
                "resolution": "video resolution",
                "format": "video format",
                "download_link": "download_link"
            }

API 3: Download Audio

        
            Request:
            {
                "key": "api key",
                "video_link": "video link",
                "quality": "quality"
            }
            
            Response (200):
            {
                "status": status_code,
                "title": "Audio file title",
                "quality": "Audio quality",
                "format": "audio format",
                "download_link": "download_link"
            }
            
            Response (400):
            {
                "status": status_code,
                "description": "reason for failure"
            }
